# uBlacklist

Aggregated blocklist for [uBlacklist](https://github.com/iorate/uBlacklist) browser extension.

## Usage

Subscribe to the raw list URL:
```
https://raw.githubusercontent.com/slashformotion/uBlacklist/main/uBlacklist.txt
```

## Sources

- [github-translation](https://github.com/arosh/ublacklist-github-translation)
- [stackoverflow-translation](https://github.com/arosh/ublacklist-stackoverflow-translation)
- [content-farm-list](https://github.com/wdmpa/content-farm-list)
- [content-farm-ai](https://github.com/agsimmons/ai-content-blocklist/)
- Personal additions in `personalList.txt`

## Build

```bash
./build.sh
```

Auto-updates daily via CI.
